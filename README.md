# 야이거

> 구두로 받은 작은 업무를 빠르게 기록하고, 잊기 전에 다시 떠올리게 해주는 Windows 데스크톱 비서 앱입니다.

![야이거 사용 흐름](docs/readme/quick-flow.gif)

## 왜 만들었나요?

업무 중에는 "서버 배포 좀 해주세요", "메일 발송 부탁드려요", "레드마인 정리해 주세요"처럼 작지만 놓치면 곤란한 요청이 자주 들어옵니다. 야이거는 복잡한 프로젝트 관리 도구가 아니라, 그런 요청을 생각하기 전에 저장하고 잊기 전에 알려주는 데 집중합니다.

핵심 원칙은 단순합니다.

- 빠르게 등록한다.
- 로컬 PC에만 저장한다.
- 주기적으로 상기시킨다.
- 업무 흐름을 방해하지 않는다.

## 화면 미리보기

### 오늘 할 일

![오늘 할 일 대시보드](docs/readme/dashboard.png)

업무 제목, 요청자, 기한을 한 화면에서 확인합니다. 검색, 완료 처리, 수정, 삭제를 메인 화면에서 바로 할 수 있습니다.

### 빠른 업무 등록

![빠른 업무 등록 창](docs/readme/quick-entry-window.png)

`Ctrl + F12` 전역 단축키 또는 하단의 빠른 등록 버튼으로 입력창을 엽니다. 제목만 입력해도 저장할 수 있고, 요청자와 기한은 필요할 때만 추가합니다.

### 설정

![설정 화면](docs/readme/settings.png)

설정 화면에서는 야이거가 사용자에게 맞게 동작하도록 기본값을 조정할 수 있습니다.

- 프로필 이름: 야이거가 사용자를 부를 때 쓰는 이름
- 빠른 등록 단축키: 백그라운드에서 입력창을 여는 전역 단축키
- 비서 성격: 친근 모드, 장난 모드, 정중 모드
- 리마인드 주기: 등록된 업무를 다시 알려주는 간격
- 알림 켜기/끄기: 메인 화면 상단 버튼에서 즉시 전환

### 알림창

![알림창](docs/readme/notification.png)

알림이 켜져 있으면 미완료 업무를 주기적으로 보여줍니다. 알림창에서는 남은 업무 일부와 전체 미완료 개수를 확인할 수 있고, 클릭하면 메인 화면으로 돌아가 업무를 정리할 수 있습니다.

## 주요 기능

- 빠른 업무 등록: 제목, 요청자, 기한, 아이콘을 최소 입력으로 저장
- 전역 단축키: 백그라운드에서도 빠른 등록 창 호출
- 메인 화면 모달: 메인 화면이 열려 있을 때 단축키를 누르면 앱 안의 등록 모달 열림
- 업무 목록: 오늘 할 일, 요청자, 기한, 완료 상태 확인
- 검색: 현재 등록된 업무를 즉시 필터링
- 완료 처리: 체크 버튼으로 업무 완료
- 수정/삭제: 등록된 업무를 바로 정리
- 주기적 알림: 잊기 전에 미완료 업무 상기
- 설정 관리: 이름, 단축키, 비서 성격, 리마인드 주기, 알림 상태 변경
- 로컬 저장: SQLite 기반으로 사용자 PC에만 데이터 저장

## 사용 흐름

1. 업무 요청을 받습니다.
2. `Ctrl + F12`를 누르거나 빠른 업무 등록 버튼을 클릭합니다.
3. 업무 제목을 입력하고 `Enter`로 저장합니다.
4. 메인 화면에서 오늘 할 일을 확인합니다.
5. 완료한 업무는 체크해서 목록에서 정리합니다.

## 데이터 저장 정책

야이거는 외부 서버, 클라우드 동기화, AI 연동을 사용하지 않습니다. 업무와 설정 데이터는 로컬 PC의 SQLite 데이터베이스에 저장됩니다.
